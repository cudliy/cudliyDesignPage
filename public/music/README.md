# Background Music Files

This directory contains inspirational background music files for the gift viewing experience.

## Files:
- `inspirational1.mp3` - Uplifting ambient music
- `inspirational2.mp3` - Gentle motivational melody  
- `inspirational3.mp3` - Soft inspirational theme

## Usage:
These files are automatically played during the image gift viewing experience to enhance the emotional impact of the shared memories.

## Note:
If these files are not available, the application will gracefully fall back to silent mode without breaking functionality.
